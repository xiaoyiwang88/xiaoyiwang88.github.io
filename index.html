Featured Projects

Argus

An interactive tool supports a Priori power analysis by exploring effect size, sample size, confounds, and statistical power, given particular experimental design configurations.  

 

RegLine

A visual analytics system facilitates iterative validation and refinement of linear regression models by exploratory data analysis (e.g., data transformation, unusual data identification).  

 

Time-series forecasting Evaluator

A visual analytics system helps model users evaluate the forecasting ability of time-series models and gain insights through structured navigation.  

 

Uncertainty Visualization

A novel design for uncertainty visualization helps users interpret uncertainty generated by predictive models and make informed decisions.  

 

Asynchronous Exergaming

This project increased the motivation of carrying out physical workouts by asynchronously linking physical exercises to computer games. Players can use their performance of prior physical workouts as power-ups during their game play.  

 

sPortal

Sportal is a digital exercise game (exergame) that motivates players to perform phycial exercises. It added motion-based control to a commercial AAA game "Portal 2", which requires players to use their movements to control the game.
